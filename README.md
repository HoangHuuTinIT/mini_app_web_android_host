# Tích hợp mini-app web vào trong app android

Tài liệu này cung cấp 2  thư mục chứa code android(host_mini_apptelegram) và mini-app(vuejs-template)

---

## 1. Clone repo

Chạy lệnh:
```
git clone https://github.com/HoangHuuTinIT/mini_app_web_android_host.git
```
---

## 2. Chạy dự án

1. Khởi động Android Studio.
2. Chọn **Open** (hoặc vào menu **File > Open**).
3. Tìm đến thư mục chứa source code Android (ở đây là host_mini_apptelegram)
4. Nhấn **OK** và đợi Gradle đồng bộ (Sync) xong.
5. Chạy dự án
